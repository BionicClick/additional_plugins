Dokumentace k pluginu 'Posílání příspěvků pomocí XML-RPC'
(serendipity_event_xmlrpc)

Dokumentaci k tomuto pluginu přeložil do češtiny Vladimír Ajgl (vlada [zavinac] ajgl [tecka] cz) a naposledy upravil dne 21.1.2012. Od té doby mohl být plugin pozměněn nebo mohly být rozšířené jeho funkce. Zkontrolujte pro jistotu i aktuální anglický ChangeLog.

Popis pluginu

Umožňuje posílat/editovat příspěvky pomocí XML-RPC API (MT, Blogger, WordPress Endpoints)

Historie verzí (ChangeLog)

  • verze 1.48
    • Opraveno mnoho chyb s kompatibilitou během testování s klientem WP Blackberry.
  • verze 1.47
    • Rozhraní vylepšeno o volání WordPress RPC
    • Do nastavení přidány informace o nastavení adresy pro volání RPC URL adresy. Plus pár dalších nastavení.
    • Do nastavení přidán adresář pro upload médií.
    • Definování spamových eventů: Signalizace hamu a spamu AntiSpamovým pluginům, které podporují příslušný event hook.
  • verze 1.45
    • Odchycení prázdných časových razítek, díky uživateli [TimNiceBut]
  • verze 1.44
    • Oprava špatného výstupu při nastavení kategorií, díky uživateli [patk]
  • verze 1.43
    • Přidána výchozí kategorie pro posílání příspěvků
  • verze 1.41
    • Oprava API volání metody mt_supportedMethods uživatelem [TimeNiceBut]
  • verze 1.38
    • Chyba #1931794: Lepší detekce base64
  • verze 1.37 [garvinhicking]
    • Chyba #1922509: Zohlednění MT API pro přílohy zakódované v base64, oprava od uživatele [Sven Herzberg]
  • verze 1.36 [garvinhicking]
    • Chyba #1904377: Zohlednění GMT, oprava od uživatele [Mike Tyson]
  • verze 1.33 [brockhaus]
    • Vytváření nevalidních (a špatných) odkazů. Změněný atribut url na správný href.
    • Při aktualizaci příspěvků bez zadání nového data bude použito staré datum místo obnovení podle aktuálního data.
  • verze 1.32 [brockhaus]
    • $serendipity['xml_rpc_default_moderate_comments'] nefungovalo :-/
  • verze 1.31 [brockhaus]
    • Hlavička byla v nětkerých situcích poslána dvakrát, což mělo za následek špatné XML odpovědi.
    • Převod HTML entit v nadpisech příspěvků na normální utf-8 znaky při ukládání příspěvků.
    • Výchozí hodnoty pro allow_comments and moderate_comments se dají nastavit v serendipity_config_local.inc.php: 
            $serendipity['xml_rpc_default_allow_comments'] = true;
