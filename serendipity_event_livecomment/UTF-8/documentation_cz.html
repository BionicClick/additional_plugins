Dokumentace k pluginu 'Vylepšené políčko komentářů'
(serendipity_event_livecomment)

Dokumentaci k tomuto pluginu přeložil do češtiny Vladimír Ajgl (vlada [zavinac] ajgl [tecka] cz) dne 13.5.2012. Od té doby mohl být plugin pozměněn nebo mohly být rozšířené jeho funkce. Zkontrolujte pro jistotu i aktuální anglický ChangeLog a credits.txt.

Popis pluginu

Používá JavaScript k ukázání náhledu komentáře a značkovací tlačítka.

Historie verzí (ChangeLog)

  • verze 2.5.4
    • Upravení jQuery skriptů, aby byly kompatibilní s jinými frameworky
    • Neodkazovat se na javascript, pokud není zobrazen jednotlivý příspěvek.
  • verze 2.5.3
    • Funkce:
      • Přidáno tlačítko URL do komentářových tlačítek
      • Přidán bublinový text při najetí na tlačítko
      • Značkovací tlačítka nyní fungují, i když není vybrán žádný text
      • Náhled lze nyní odznačit
    • Výkon:
      • Redukce počtu HTTP požadavků
      • Redukce velikosti JavaScriptových souborů
      • Redukce počtu procházení DOM
      • Ajaxové volání se už neprovádí dvakrát, pokud je jsou aktivována značkovací tlačítka i náhled
      • Ajaxové volání používá metodu GET místo POST
      • Přidána volba pro vložení javascriptu jako parametru
      • Javascript se zastaví, pokud není žádný formulář pro komentář
      • Javascript přesunut do patičky
      • Přidána volba k zadání cesty, takže již není nutné ji zjišťovat dynamicky
    • Různé:
      • lcnewbase se nyní inicializuje na správném místě
      • Změn způsobu použití globálních proměnných
      • Možnost použití značkovacích tlačítko, i když se používá jiný než standardní adresář pro pluginy
      • Aktualizován jquery.elastic na verzi 1.3, který opravuje chybu s rámečkem v Opeře
