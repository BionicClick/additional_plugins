Dokumentace k pluginu 'EMAIL_BOT_OBFUSCATOR'

Dokumentaci k tomuto pluginu přeložil do češtiny Vladimír Ajgl (vlada [zavinac] ajgl [tecka] cz) dne 12.4.2010. Od té doby mohl být plugin pozměněn nebo mohly být rozšířené jeho funkce. Zkontrolujte pro jistotu i aktuální anglickou dokumentaci.

Plugin má dvě hlavní vlasnosti:

Obsah

  1. Klikací mail
  2. Obfuskace (zmatení) emailové adresy

Klikací mail

Do příspěvku nebo statické stránky můžete napsat mailovou adresu jako email@priklad.cz bez jakéhokoliv html kódu kolem něj. Plugin emailovou adresu vyhledá a přidá kolem ní html tagy, aby byla přístupná po kliknutí, například takto: <a href="mailto:email@priklad.cz">email@priklad.cz</a>

Obfuskace (zmatení) emailové adresy

Mnoho lidí se (oprávněně) bojí robotů, které po internetu hledají emailové adresy, na které pak posílají tuny spamu. Plugin umožňuje znesnadnit práci těmto robotům dvěma způsoby. Dopředu byly zavrženy špatné možnosti jako obrázek s textem mailu, které zhoršují dostupnost webu (například pro nevidomé lidi). Tedy použité možnosti znesnadnění emailové adresy jsou následující:

  1. složení adresy pomocí javascriptu:
    <script type="text/javascript">var username = "mail"; var hostname = "priklad.cz";document.write("<a href=" + "mail" + "to:" + username + "@" + hostname + ">" + username + "@" + hostname + "<\/a>")</script>
  2. Podle mého je ještě lepší následující varianta využívající html entity. Každý prohlížeč je zobrazí normálně, ale roboti budou mít větší problémy:
    <a href="mailto:&#101;m&#97;i&#108;&#64;prik&#108;ad#46;cz">&#101;m&#97;i&#108;&#64;prik&#108;ad#46;cz</a>
  3. a nebo nedělat nic
